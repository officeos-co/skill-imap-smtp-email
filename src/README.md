# IMAP/SMTP Email

Read, search, manage, and send email via IMAP and SMTP through a proxy service. Works with Gmail, Outlook, Fastmail, and any standard mail server.

## Categories

- Productivity

## Credentials

| Key | Label | Kind |
|-----|-------|------|
| `proxy_url` | Proxy URL | url |
| `imap_host` | IMAP Host | text |
| `smtp_host` | SMTP Host | text |
| `email` | Email Address | text |
| `password` | Password / App Password | password |

## Installation

This skill is included in [EnterpriseAgentOS](https://officeos.co) and requires no separate installation.
